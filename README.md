## Run Instructions
On windows, you can use ```pip install --user pipenv``` to install pipenv
1. [Install](https://pipenv.pypa.io/en/latest/install/#installing-pipenv) [pipenv](https://pipenv.pypa.io/en/latest/)
1. ```pipenv install```
1. ```pipenv shell```
1. ```python main.py```
